# Multilingual Loan Disbursal System

The Multilingual Loan Disbursal System is a comprehensive web application designed to simplify the process of exploring, comparing, and applying for loans. Built with a modern technology stack, it offers a suite of powerful tools for users, from an interactive dashboard and EMI calculator to an AI-powered banking advisor. The platform is fully internationalized, supporting over 25 languages, and provides robust user authentication and data management through Firebase.

## Key Features

*   **Interactive Dashboard**: A central hub displaying key loan metrics, market statistics, and visual charts for interest rates and bank comparisons.
*   **Market Comparison**: Users can browse and filter a comprehensive list of loan products from various banks, sorting by interest rates, tenure, and more.
*   **Compare & Calculate**: A powerful tool to calculate Equated Monthly Installments (EMI), view detailed amortization schedules, and compare different loan scenarios side-by-side.
*   **AI Banking Advisor**: An intelligent chatbot that assists users with their banking and loan-related questions, guiding them toward suitable loan types based on their needs.
*   **Loan Application System**: A multi-step form that allows authenticated users to fill out and submit detailed loan applications, which are saved securely to their profile.
*   **User Profile & Authentication**: Secure user sign-up, login, and profile management is implemented using Firebase Authentication. Users can view and manage their submitted applications.
*   **PDF Report Generation**: Users can download a detailed PDF summary of their loan calculations and application data for their records.
*   **Multi-Language Support**: The entire application is internationalized, with support for over 25 languages to cater to a global audience.

## Technology Stack

*   **Framework**: Next.js (with App Router)
*   **Language**: TypeScript
*   **Styling**: Tailwind CSS
*   **UI Components**: ShadCN/UI
*   **Backend & Database**: Firebase (Authentication & Firestore)
*   **Generative AI**: Genkit
*   **Charts & Visualizations**: Recharts
*   **Form Management**: React Hook Form & Zod for validation
*   **Internationalization (i18n)**: Custom solution using React Context.

This project was bootstrapped and developed in Firebase Studio.
